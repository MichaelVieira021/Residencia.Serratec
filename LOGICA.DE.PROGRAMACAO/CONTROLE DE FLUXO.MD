# CONCEITUAÇÃO
</div>
- São estruturas que tem a capacidade de direcionar o fluxo de execução do código.

<br>

# TIPOS 

- DECISÃO:  
  - if             -(se)
  - if-else        -(senão)
  - if-else-if     -(senão se)
  - switch         -(escolha)
  - operador ternário

- REPETIÇÃO:
  - for             -(para)
  - while           -(enquanto)
  - do while        -(faça enquanto)

- INTERRUPÇÃO:
  - break           -(pare)
  - continue        -(continua)
  - return          -(retorne)
