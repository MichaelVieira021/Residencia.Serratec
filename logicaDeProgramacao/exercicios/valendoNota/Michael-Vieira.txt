A) Pesquise e responda:

1) O que é algoritmo?
    Algoritimo é um passo a passo para resolver um problema.

2) O que é um programa de computador?
    Um programa é um conjunto de algoritimo que resolve um problema.

3) O que é uma variável?
    Uma variável é utilizada para armazenar dados.

4) O que é o Portugol Studio?
    PortugolStudio é uma IDE utilizada para produzir pseudocodigo.

5) O que faz o comando escreva do Portugol?
    Exibe uma informação no terminal.

6) O que faz o comando leia do Portugol?
    Solicita a entrada de um dado.

7) Defina o que é tipo de variável
    Um tipo de variável define o tipo de dado que pode armazenar e as operações que podem ser realizadas sobre eles.

8) Explique o que são os tipos de variáveis do Portugol:
   a. cadeia
    Conjunto de caracters.

   b. inteiro
    Número inteiro.

   c. real
    Número com casas decimais.

   d. logico
    verdadeiro ou falso.

   e. caracter
    Carácter único.
